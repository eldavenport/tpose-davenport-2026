# tpose-davenport-2026

[![DOI](https://zenodo.org/badge/956241578.svg)](https://doi.org/10.5281/zenodo.21087334)

This project contains code to generate the figures in Davenport et al. 2026, 'Vertical Momentum Flux at 0N, 140W: Unresolved Internal Waves and Implications for SST'. Output from the Tropical Pacific Ocean State Estimate is available at [this link](https://ecco.ucsd.edu/tpose.html).
